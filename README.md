一个粗陋的网站，记录一些贤狼赫萝的有趣语录。

站点地址：[https://blog.lichenrobo.com/horo/?page=1](https://blog.lichenrobo.com/horo/?page=1)

----------------

## Features

- 收录赫萝语句的中日双语文本及章节出处；
- 每天对语句随机排序；
